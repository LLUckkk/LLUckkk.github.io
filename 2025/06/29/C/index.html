语法

  • 增加了bool类型：true、false

  • 输入输出：

    endl：换行符

  • lambda表达式：[capture](parameters)->return-type{body}

  • 字符串类型：String

  • 关于引用：

    • 在使用的时候，r可以就是i的值，不用解引用

    • 不存在空的引用

    • 一旦引用被初始化后就不能再指向别的对象

    • 引用必须在创建的时候就进行初始化

STL

主要分为六大模块：容器、算法、迭代器、仿函数、适配器、分配器

容器和算法通过迭代器进行衔接：迭代器可以粗浅的理解为指针，能够通过迭代器对容器中的元素进行操作

容器

对于容器，有序列式和关联式：在于是否需要关注存储时的数据顺序

vector

动态数组，且位于堆空间而非栈空间，因此局部区域开大的vector不爆栈。

String

deque

对头部的插入和删除速度会比vector快。

stack

栈，只有头部元素可以使用，后进先出

queue

队列，先进先出

list

链式存储，可对任意位置快速插入or删除

迭代器

  • iterator：正向迭代器
  • const_iterator
  • reverse_iterator：反向迭代器
  • const_reverse_iterator
