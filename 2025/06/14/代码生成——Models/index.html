代码生成——Models

Created2025-06-14|Updated2025-06-16
|Post Views:

本篇文章总结代码生成论文看到的较为经典的模型

Repo

repo级别的pos embedding：ALiBi，RoPE，允许更长的seqence

model

  • RepoCoder：

    利用检索器定位仓库内有价值的信息，建立仓库内的检索数据库，然后将需要补全的代码做检索查询，提取最相关的代码片段。将检索到的信息和需要补全的代码放入提示词模版进行生成。

  • CoCoMIC：跨文件

  • RepoHyper：语义图结构

  • RLFG：设置了RLPG框架，从整个代码仓库中提取上下文信息，生成特定的提示。

  • Repoformer：selective RAG framework

  • RepoFusion：combine relevant contexts，相比较RepoCoder对筛选的信息和prompt更为精简，基于相关性引导的上下文选择策略（Relevance-Guided Context Selection）指导模型Prompt的构建

  • CodePlan：planing problem

  • A^3-codgen：

    将代码仓库中的三种关键信息整合到代码生成过程中：局部信息、全局信息、第三方库信息（针对python语言）

    • 局部信息：当前代码文件中的上下文信息，例如函数签名、变量定义及模块的路径
    • 全局信息：代码库中其他文件中的函数及其依赖关系。
    • 第三方库：代码库中预先安装的第三方库。

benchmark

  • RepoEval
  • Stack-Repo
  • Repobench
  • EvoCodeBench
  • SWE-bench
  • CrossCodeEval
  • SketchEval

CodeT5

motivation

  • encoder-only或者decoder-only
  • 忽略了code本身的special characteristics

contributions

  • unified framework to support both code understanding and generation，both encoder and decoder
  • identifier-aware pre-training：
    • distinguish which tokens are identifiers and recover them when masked
  • exploit comment for bimodal dual generation task：learn a better NL-PL alignment

method

  • based on T5 architecture
    • seq2seq
    • derive generic representations for PL and NL
  • pre-train codeT5 on CodeSearchNet
    • input：PL-only or NL-PL
    • tasks：masked span prediction、identifier tagging、masked identifier prediction
  • fine-tune Code-T5 on most tasks in the CodeXGLUE ：
    • understanding task：defect detection、clone detection
    • generation task：code summarization、generation、translation、refinement
    • pre-training：span prediction；fine-tuning：complete code
  • identifier tagging and prediction tasks
    • convert PL segment into AST
    • extract node types
  • bimodal dual learning objective
  • task control codes：Summarize、Generate、Defect、Refine、Translate

CodeBERT

motivation

首个面向多编程语言的双模态预训练模型

method

  • RoBERTa-base
  • 双模态数据和单模态数据
  • pre-train：MLM和replaced Token detection
  • 输入支持：NL描述 + Code上下文 + AST结构

CodeGen

motivation

文本-代码混合预训练

method

decoder noly

  • 大规模自然语言文本进行预训练
  • 自然语言+代码混合数据继续训练
  • 纯代码语料微调

Benchmark & Metrics

  • HumanEval
    • Python函数生成：函数注释+单元测试
    • pass@k
  • CoderEval
    • HumanEval 的扩展
  • CodeSearchNet
    • 自然语言函数描述 + 真实代码实现
    • 代码检索（code search）、代码生成
  • MBPP
    • Python 基础编程题：自然语言描述、参考实现和 3 个测试用例
    • 模型对基础算法题的泛化能力。
