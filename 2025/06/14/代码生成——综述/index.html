代码生成——综述

Created2025-06-14|Updated2025-06-15
|Post Views:

综述

task

  • text2code
    • code retrieval：BERT similarity
    • code synthesis
    • text2SQL
    • Math progarmming
  • code2text
    • code summary
    • code review
    • identifier prediction
  • code2code
    • code search
    • code completion：ngram、RNN、Transformeer
    • code translation：one2anotherlanguage，seq2seq
    • code repair：fix bugs，seq2seq
    • cloze test：keywords
    • code infilling：左右上下文都给出
    • unit test generation：unitest for LLM’s program
    • type prediction
  • code2patterns
    • defect detection
    • clone detection
    • code classification
    • code reasoning：

evaluate

  • pass@k：对一个问题生成k次，至少一次能通过测试的概率
  • HumanEval
  • APPS
  • MBPP
  • 数学方面：MathQA-python、GSM8K-Python
  • repo级别：RepoBench，SWE-bench

模型前面的多少多少B表示的参数量，也即billion，以十亿为单位

  • CodeSearchNet
  • CoderEval

model

  • CodeT5
    • T5 original span corruption
    • identifier tagging
    • masked identifier prediction
  • AlphaCode
  • CodeGen
  • Pangu-Coder
  • CodeGeeX

使用AST/IR

  • InterCode
  • TreeBERT
  • SynCoBERT：使用深度优先搜索处理AST然后连接到code和comment中
  • UniXcoder：把处理后的AST作为输入而不是源码

缺陷：静态，没有捕捉到源码在运行时的一些特性

控制流和数据流

  • GraphCodeBERT：data flow + CodeBERT

记录

  • project的瓶颈：窗口相比较project的长度太小
