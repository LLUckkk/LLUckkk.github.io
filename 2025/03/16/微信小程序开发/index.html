微信小程序开发

Created2025-03-16|Updated2025-03-17
|Post Views:
  • 小程序和web开发的区别：
    • 运行环境不同（微信环境/浏览器）
    • API不同（小程序无法调用DOM和BOM的API，不过小程序可以调用微信提供的API）
    • 开发环境不同（小程序需要申请小程序开发账号，安装小程序开发工具，并创建和配置小程序项目）
  • 微信开发环境所提供的工具：组件、扩展能力、接口、云开发
  • 微信开发者工具是官方推荐使用的小程序开发工具
  • 小程序有两种成员角色：项目成员和体验成员
  • 小程序ID：小程序在微信账号体系内的唯一凭证（新建小程序项目、真机调试、发布小程序的时候必须使用）
  • 小程序密钥（微信登录、支付、发送消息等高级开发）
  • 微信开发者工具必须联网使用

文件目录结构

一个完整的小程序项目分为两个部分：主体文件、页面文件

主体文件

主体文件又称为全局文件，能够作用于整个小程序，影响到小程序的每个页面，主体文件必须放到项目的根目录下

主体文件由三部分组成：

  • app.js：小程序入口文件
  • app.json：小程序全局配置文件
  • app.wxss：小程序全局样式

主体文件必须以app开头

页面文件

页面文件是每个页面所必需的文件，小程序的页面文件都放在pages目录下，一个页面一个文件夹

每个页面通常有四个文件组成

  • .js页面逻辑
  • wxml页面结构
  • wxss页面样式
  • json页面配置

js和wxml是必需的

可以通过app.json和pages新建文件夹两个方法新建页面

component

存放所有页面的公共组件

配置文件

JSON是一种轻量级的数据格式，常用于前后端数据交互，不过在小程序中JSON用来配置项目或者页面的属性和行为，每个页面或者组件也都有一个对应的JSON文件

全局

app.json：全局配置文件，配置小程序的一些全局属性和页面路由

  • pages：指定小程序由哪些页面组成

    调整页面顺序，新增或删除页面都需对其进行操作

  • window字段：设置小程序的状态栏、导航条、标题、窗口背景色

  • tabBar字段：定义小程序顶部、底部tab栏，用以实现页面之间的快速切换

页面

页面.json：配置当前页面的窗口样式、页面标题等

页面配置选项和全局配置类似，当页面配置和全局配置冲突时，页面配置优先。

项目配置

project.config.json：小程序项目的配置文件，用于保存项目的一些配置信息和开发者的个人设置

private的配置文件配置个人的配置，另一个配置公共配置。因此要注意把private的文件放置到.gitgnore文件中，而把与最终编译结果有关的设置设置到project.config.json中

sitemap

配置小程序机器页面是否允许被微信索引，提高小程序在搜索引擎被搜索到的概率

一般不需要配置，默认所有页面都可以检索

小程序样式

wxss仅支持部分选择器，新增了尺寸单位rpx，标签不能使用html的标签（参考官方文档）

rpx

解决适配不同屏幕宽度的手机的问题，可以根据不同设备的屏幕宽度进行自适应缩放，规定任何型号的手机屏幕宽度都是750rpx

在iPhone6下，px和rpx的换算关系是：1rpx = 0.5px

全局样式

app.wxss，设置字号、背景色、宽高等全局样式

局部样式

页面.wxss，局部页面样式，优先级更高

组件

view

小程序提供的容器组件，可以直接当成div使用

swiper

swiper-item实现轮播图

swiper是轮播图容器

swiper-item是轮播图的每一项

image

  • src
  • mode：
  • lazy-load：懒加载，在滑动到上下三屏以后展示图片
  • show-menu-by-longpress：长按显示菜单，分享，识别，收藏，保存等

默认宽度320px，高度240px

text

渲染文本

  • user-select：文本是否可选
  • space：连续空格
    • nbsp：字符大小空格
    • emsp：中文空格
    • ensp：英文空格
  • text组件内只能嵌套text组件

navigation

实现跳转功能

  • url路径后可以携带参数，注意参数和路径之间？分隔，参数键和值用=相连，不同参数用&分隔

    在onLoad生命周期函数中获取传递的参数

  • navigate：只跳转非tabbar页面，保留上一级页面

  • redirect：只跳转非tabbar页面，关闭上一级页面

  • switchtab：只跳转tabbar页面，关闭其他非tabbar页面，不能携带参数

  • reLaunch：关闭所有页面，然后打开某个页面

  • navigaBack：返回上一页或者上几页
