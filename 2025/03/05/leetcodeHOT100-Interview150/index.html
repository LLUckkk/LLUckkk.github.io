  • cmp的易错点：

    • 返回值小于0说明第一个元素排在第二个元素之前

    • 返回值大于0说明第一个元素排在第二个元素之后

    • 两个const void参数是指向数组元素的指针。

LRU

其实之前学过LRU最先最少使用替换，但是没学过代码实现。

难点在于如何实现链表的O(1)时间复杂度的插入和查找。题目把结构体的定义也空出来了，说明可以在链表结构上下功夫——使用双指针链表+hash

注意这里并不需要真的记录访问了多少次，而是通过链表操作，每次将访问的或者新加的节点放在链表的头部即可，然后当删除节点的时候直接删除尾部节点就可以

仍然是需要注意删除头部尾部节点的边界情况

最长公共子序列

原来都是写伪代码，转化为实际代码的时候注意下标，i-1和j-1

状态转移方程是：

  • dp[i][j]表示str1前i个字符和str2前j个字符的最长公共子序列
  • 当str1[i]和str2[j]不等时：dp[i-1][j]和dp[i][j-1]的最大值
  • 当相等时，dp[i-1][j-1]+1

每日温度

这个比较简单，用一个值栈和一个下标栈，从头扫描温度。

当栈为空，数字和下标分别进栈；否则，比较当前温度和栈顶温度，大于栈顶温度则将栈中的温度弹出直到为空或者栈顶温度大于当前温度，用当前温度下标减去弹出的温度的下标即为对应位置的ans

最后，栈中剩余的温度对应的ans都为0。

49.字母异位词分组

  • hashfunction返回unsigned int的原因是：

    防止int在字符串非常大的情况下溢出变为负数，而后续这个返回值要做数组下标，如果是负数的话会报错

    并且能表示的范围也更大了

  • 常见的字符串哈希方式：

    • BKDR Hash

    • DJB2哈希

      但更容易产生哈希冲突

    • SDBM

      适用于数据规模较大的情况，hash冲突少，但计算慢

  • 几个新函数

    申请size个元素的空间，且每个元素置0或者NULL

    将原来申请的ptr空间扩展为newSize

    相当于先malloc在strcpy

  • 关于本题的思路：

    • 定义结构体HashNode，存储每个hash值对应的字符串
    • 遍历字符串数组，将每个字符串内的字母排序
    • 排序后在hashtable里寻找其对应的HashNode，没有则新建节点插入
  • /**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume
 * caller calls free().
 */

#define HASH_SIZE 50000

typedef struct HashNode {
    char* key; //排序后的字符串作为键值
    char** words; //在这个键值下的字符串
    int cnt; // 这个键值下有多少个字符串
    int capacity; //其实没太懂这个capacity
    struct HashNode* next; //也没太懂
} HashNode;

unsigned int hashFunction(char* str) {
    unsigned int hash = 0;
    while (*str) {
        hash = hash * 31 + (*str++);
    }
    return hash % HASH_SIZE; //计算hash值
}

int cmp(const void* e1, const void* e2) {
    char* p = (char*)e1;
    char* q = (char*)e2;
    return (*p) < (*q);
}

void sortString(char* str) { qsort(str, strlen(str), sizeof(str[0]), cmp); }

HashNode* Find(HashNode** hashTable, char* key) {
    unsigned int index = hashFunction(key);
    HashNode* node = hashTable[index];

    while (node) {
        if (strcmp(node->key, key) == 0) {
            return node;
        }
        node = node->next;
    }

    node = (HashNode*)malloc(sizeof(HashNode));
    node->key = strdup(key);
    node->cnt = 0;
    node->capacity = 2; //这个就是随意设置的
    node->words = (char**)malloc(sizeof(char*) * node->capacity);
    node->next = hashTable[index];
    hashTable[index] = node; //将新建的node查到头部
    return node;
}

char*** groupAnagrams(char** strs, int strsSize, int* returnSize,
                      int** returnColumnSizes) {
    HashNode** hashTable = (HashNode**)calloc(HASH_SIZE, sizeof(HashNode*));
    *returnSize = 0;

    for (int i = 0; i < strsSize; i++) {
        char* sorted = (char*)malloc(sizeof(char) * 105);
        strcpy(sorted, strs[i]);
        sortString(sorted);
        HashNode* node = Find(hashTable, sorted);

        if (node->cnt >= node->capacity) {
            node->capacity = node->capacity * 2;
            node->words =
                (char**)realloc(node->words, node->capacity * sizeof(char*));
        }

        node->words[node->cnt++] = strs[i];
    }

    char*** res = (char***)malloc(sizeof(char**) * strsSize);
    *returnColumnSizes = (int*)malloc(sizeof(int) * strsSize);

    for (int i = 0; i < HASH_SIZE; i++) {
        HashNode* node = hashTable[i];
        while (node) {
            res[*returnSize] = node->words;
            (*returnColumnSizes)[*returnSize] = node->cnt;
            (*returnSize)++;
            node = node->next;
        }
    }
    return res;
}
